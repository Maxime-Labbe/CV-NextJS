## Installation

To get started with this project, clone the repository and install the dependencies:

```bash
git clone https://github.com/Maxime-Labbe/CV-NextJS
cd CV-NextJS
npm install
```

## Usage

To run the project locally, use the following command:

```bash
npm run dev
```
